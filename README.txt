CS351: Databases
University Management System

This project is a university database system developed in sqlite 3 and Java in Eclipse (os was Mint 13 and Fedora 19). To run, unzip the project folder from the tar ball import it in eclipse. The sqlite java driver is required for the project, and was added to it at the time of development. If the project doesn't run initially this will need to be added to your version of eclipse.
